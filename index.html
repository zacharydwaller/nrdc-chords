About This Project

The goal of this project is to create an open-source near-realtime interface between the Nevada Research Data Center (NRDC) and CHORDS a data visualization package. While the product is primarily being built for the NRDC, there will be an emphasis on building the project to be as generalized and generic as possible. The goal end product will be a plug-and-play solution that will allow any small or medium sized science team to take advantage of the benefits of data visualization at no cost and limited technical knowledge. An open-source approach will allow for wide adaptability and customization of the product and be offered as a free alternative to existing proprietary or custom-built data visualization solutions.

We are working with the Nevada Research Data Center (NRDC) to develop an open-source interface between arbitrary data streams and CHORDS, a data visualization package. In addition to fulfilling the NRDC’s requirement that the package work with their cyberinfrastructure, there will be an emphasis on building the project to be as generic as possible. The goal end product will be a plug-and-play cloud-hosted visualization package that can interface with any variety of standard data streams to visualize the data in near-real time. This will allow any small to medium sized scientific research team to take advantage of the benefits of data visualization with little effort and no cost.

Data visualization is an important tool for scientists working with large and constantly updating data sets. However, scientists are usually presented with two options for robust visualization: expensive proprietary solutions, or programming language libraries that require developer-level knowledge to use. Our project will benefit scientists by giving them the full benefits of data visualization in a free and easy to use package. This will save science teams the cost of integrating proprietary solutions and also enable teams with little-to-no technical knowledge to use our product. In addition, our project will also be open source, which will allow teams with the technical skill to fork the source code and build their own custom solution using our platform.
